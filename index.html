Documentation

Usage is really simple! Just add an 'input' field to your UI next apply in JavaScript the 'tagsInput' plugin. You can use whatever JQuery selector to select to which input field to apply the tags! 

		$(document).ready(function() {
			$('#tags').tagsInput({

			});
		});
Tags can be either plan text, the entire input string is then something like: tag1;tag2;tag3;.
But tags can also start with a color definition, eg: #FF4422tag1;#553311tag2;#22FF54tag3; or a combination: #FF4422tag1;tag2;tag3;. If no color is specified the default color will be used that is specified to the 'tagsInput'-options, or if non is specified it will be #36454f.
Following options are available:
  • onChange: A method that will be triggered upon every change (add, remove, color change) of any tag. It submits the entire form-input-value after the change is applied.
  • defaultLabelColor: The default color of a newly added label or of a label without color.
  • allowColorChange: true|false indicating if the color-change caret should be available or not.
  • hints: String[] containing the hints will be shown with their color (if they have one, otherwise the default color from defaultLabelColor) from which you can select one to easily add to your tags.
  • maxHints: integer defining how many hints will be shown at maximum. Default is set to 5
  • hintsMaxWidth: integer (pixels) defining the maximum width of the hints dropdown. If the matching input field is smaller then the dropdown will take the width of the input. Default is set to 300
  • disabled: Adding the attribute to the form control disables the entire tag input (read-only mode).
